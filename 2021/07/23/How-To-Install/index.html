    • 说在前面

    最近一直在沉迷玩3ds的游戏，但是装cia却是一个麻烦事…

    用FBI安装的话…又慢而且占的空间还大，实在是没有这样的耐心…

    所以我开始在网上寻找教程，在老男人的论坛上找到了方法并且安装成功了

    但是原帖排版有点乱，所以就写下了这个教程，顺便也帮助一下同样需要的人

    废话有点太多了

    所需要的物品…

    • 一台b9破解的3ds

    • 一台装有python的电脑

    • 想装的cia文件

    1.下载custom-install 和seedconv工具并解压

    如果觉得下载慢没有魔法上网的话，我也提供了蓝奏云的地址，密码是2xuj

    2.在3ds中提取出必要的文件

    首先在3ds关机状态下同时按下Start+开机键，并选择第一个进入gm9

    在进入gm9后，找到sysnand ctrnand/private/movable.sed，如图所示

    之后按A，在下屏选择copy to 0:/gm9/out将文件导出

    之后到sysnand ctrnand/data/一串英文/sysdata/001000f/00000000

    还有memory virtual/boot9.bin中导出这两个文件，步骤和上面一样

    导出之后这些文件都会在sd卡根目录\gm9\out里，一会儿安装要用

    3.将0000000转为seeddb

    把3ds的内存卡插到电脑上，在目录中找到刚刚导出的三个文件中找到0000000

    然后把他拖到seedconv.exe上（真的就是拖到那里，不用打开）

    之后将会出现seeddb.bin是我们需要的文件，也将他放到一个你习惯的地方

    4.安装cia

    打开刚刚解压的custom-install文件夹，把custom-install-finalize.3dsx拖到sd卡的根目录

    打开ci-gui.exe

    在SD root的那一栏选择自己3ds的sd卡根目录

    后三栏填刚刚导出的三个同名文件就行（如果都在sd卡根目录就会自动填写）

    在Add CIAs里找到你需要安装的CIA

    点击Start Install

    5.Enjoy(～￣▽￣)～
