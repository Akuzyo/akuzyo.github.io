关于html里面会有什么东西

  • 2021-07-28
  • 作者 Akuzyo
  • ~2.35K 字
  • 次阅读
本文最后编辑于 前，其中的内容可能需要更新。

我也只是一个萌新，这个冗长的笔记也是给自己写的，如果有什么不正确的地方希望能帮我指正，我也能学到很多东西ヾ(≧▽≦*)o

在了解html里面有什么东西之前，我写了一个很简单的例子..

可能看到这里的话就猜得差不多了，如果没猜到的话那就继续往下看吧

该有什么 ？

首先，我在这个例子的最前面写了一个<!DOCTYPE html>的东西。专业点来说叫做html5标准网页声明(这个名字了解下就好)。主要是用来告诉浏览器“要用这个标准来加载网页。而不是用浏览器自己的标准加载”的意思(笑)。以防止不同的浏览器渲染出不同的东西。所以在文档前写这个就显得十分有必要了（（

当然，我前面提到说这个是“html5”的声明，html5以前也是有这样的声明的，但有点太繁琐再加上自己也不会太长了就不去细说了，感兴趣的可以看看下面折叠的部分

在这里在这里OWO

1、严格模式：<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

2、过渡模式：<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

3、基于框架的 HTML 文档：<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">

所以说还是html5的网页声明简单啊_(:з)∠)_

<标签>…..</标签>

本来是想说先把<html>...</html>或<head>...</head>那些的都介绍一遍的…

但是这里想插一下标签(或者元素)的概念所以我就不用再开一个文档写这些东西了

上面这个实例中，可以看到很多的”< >”，这些可以叫做是html的标签。可以说是这些标签构成了这个这个实例

我是这么理解的：因为html是一种标记语言，所以里面会有很多的标签来“描述”这个网站也很合理

开放标签和闭合标签

标签一般都会成对出现，就像套壳一样（比如<p>...</p>）

其中第一个标签叫做开始标签（开放标签），第二个标签则为结束标签（闭合标签）

标签的属性

大多数标签是能加入属性的，大概是name=“value”的格式名字=“一个值”

例如<h6>内容</h6>的意思是标题…（就像md里的###### 内容）

但是我们可以在标签里加入属性来让他改变对齐方式

如果加入属性的话<h6 align="center">你好OWO</h6>就会成为这个下面这个样子

你好OWO

（我记得我的自我介绍里也用过这个东西…但是当时完全没有深究…( ＿ ＿)ノ）

空内容

有的标签（例如<br /> ）具有空内容，空元素会在开始标签中进行关闭（说人话就是能在第一个标签里面使用</>）

所以标签大概的格式应该是这个样子的

<标签+属性>内容</标签>

所以明白了标签的概念，可能下面要说的东西理解起来会简单一些吧╰(￣ω￣ｏ)（至少我觉得简单了不少）

html标签

继续看可以看到我在的<!DOCTYPE html>下面写了<html>的标签

而且文档的末尾也写了一个

所以说这个标签是用在整个文档上的…

就像书信的开头和落款一样，没有这个就不知道是书信了…

head标签

下面的head标签…顾名思义，就是html文档的头部标签可以引用脚本、指示浏览器在哪里找到样式表、提供元数据等

而里面的标签，同样也是顾名思义，是用来写网站的标题的,所以只能在head里..

（我也不知道该怎么描述就照搬w3schools上面的了…）

body标签

<body>标签表示了一个网站的主要部分

也就是一个网站的主体废话在这里面就是你大放文采的地方了

可以这样理解，我们要让内容在IE浏览器中显示，当然不能任意到处放，这里html规范规定有它的html语言语法，这个时候body标签解决了这样要求与规范

（例如我在上面写的<p>就是表示一个段落）

所以一个简单的html的文档里面会有的东西就是这些了…当然不只有这些，之后的笔记会慢慢地写到的（主要是还没学…( ＿ ＿)ノ）
